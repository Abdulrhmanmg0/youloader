# YouTube Downloader Pro

A modern GUI YouTube downloader built with Python and PySide6.  
Supports MP4/MP3 downloads, video quality selection, progress bar, and thumbnail preview.

## Requirements

- Python 3.10+
- PySide6
- yt-dlp
- Pillow
- requests
- FFmpeg (for MP4 downloads)
- Node.js (for certain YouTube videos)

## Run from source

```bash
git clone https://github.com/yourusername/YouTubeDownloader.git
cd YouTubeDownloader
pip install -r requirements.txt
python youtube_downloader_qt.py
